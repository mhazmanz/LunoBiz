# LunoBiz

Windows single-file Python application for:
- Market monitoring
- Extensive backtesting + walk-forward validation
- PAPER trading by default
- Controlled LIVE trading after gates pass

## Setup (Windows)
1) Install Python 3.12
2) Create virtual environment:
   - `py -3.12 -m venv .venv`
   - `.\.venv\Scripts\Activate.ps1`
   - `python -m pip install --upgrade pip`

## Secrets
- Copy `.env.example` to `.env`
- Fill `.env` with your real credentials
- `.env` is git-ignored
